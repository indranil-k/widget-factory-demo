Customized offcanvas

These customization options for the offcanvas widget offer user benefits by allowing them to:

  1. Customization 1 : Control Width - Users can set the width of the offcanvas, accommodating different content sizes or personal preferences.
  2. Customization 2 : Animation Duration - The animationDuration option enhances the user experience by specifying how quickly the offcanvas opens and closes, ensuring a smooth and visually pleasing transition.
  3. Customization 3 : Interaction with Events - Users can define their own event names (openEvent and closeEvent) for triggering offcanvas actions, providing flexibility for integrating the widget into their specific application workflows or user interactions.
